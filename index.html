And my quest lately

I've been working on an open-source project called Music Blocks, since February, 2020. Music Blocks is a Visual Programming Language for creation of music and art, using concepts of mathematics and programming.

Recently, I've been leading the development of the second generation of the application — Music Blocks 2.0 to employ a robust, modular architecture, with better interaction design and newer features.

For quite some time, I've been exploring "Media Art" and "Creative Coding" in an attempt to discover newer patterns and designs in nature.

This has been feeding my hunger for geometrical shapes, motifs, and tesellations. And, it has been educating me in discovering interesting applications of mathematical constructs.

One of my hobbies has been to experiment in creative ideas with mathematical equations and programming tools. I've created several art pieces with just mathematical curves.

I discovered some very interesting UI tools — Processing, p5, matter.js, liquidfun, etc. And, it has been very absorbing to create cool projects. I'm planning an application to simulate dynamics for high school physics.

Into programming

I got into programming at 11, with Visual Basic. It took me several attempts at learning it enough by eavesdropping older people's classes (that was the only resource I had back then) to build a barebone GUI calculator application.

I started learning Java in school since I was in the 8th grade, and kept writing school level Java programs until the end of school at 18.

In grade 11 and 12, our computer teacher used to give a few of us separate class assignment questions than the others. Those were really challenging for us back then. Learnt about some standard Data Structures and Algorithms there.

I wasn't particularly interested in solving abstract programming questions to run in a black & white terminal window. I wanted to build applications people could interact with.

In the pursuit of building applications, I taught myself quite a few programming languages and tools. Over the years I've written some good amount of code in those, to build actual projects.
